## Atividade 3
Para rodar o projeto, é necessário entrar na pasta desejada e rodar o seguinte comando para compilar o arquivo obj.cpp:
```
g++ -std=c++11 -o nomeExecutável obj.cpp
```
Para ler o .obj, basta rodar o executável:
```
./nomeExecutável
```
